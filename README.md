# Calendar_ConverterFST
Calendar Converter — десктопное приложение для Windows, которое преобразует расписание Факультета социальных технологий (Excel .xls/.xlsx) в CSV для импорта в Google Календарь. Автоматически определяет группы, фильтрует занятия, обрабатывает время и практики. Не требует установки Excel или Python. Просто, быстро, удобно.
