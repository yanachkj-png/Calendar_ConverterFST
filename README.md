# Calendar Converter by Zhekos Abrikos

**Программа для преобразования расписания Факультета Социальных Технологий (Excel-таблица из группы ВК) в CSV-файл для импорта в Google Календарь.**

---

## 📌 Функционал

- Поддерживаются файлы `.xls` (старый формат Excel, используется деканатом) и `.xlsx`.
- Автоматически определяет все учебные группы из файла.
- Позволяет выбрать нужную группу из списка.
- Фильтрует расписание только для выбранной группы, включая общие занятия.
- Обрабатывает типы занятий:
  - Лекции → добавляется префикс **«Л: »**
  - Практические занятия → префикс **«ПЗ: »**
- Правильно разбирает время:
  - Диапазоны (например, `12.10-13.40`) → точное начало и конец.
- Многодневные события (практика с периодом в скобках) создаются как целодневные.
- Место проведения:
  - Если в поле «Ауд» указано «СДО» → место = **«СДО»**.
  - Иначе → фиксированный адрес факультета.
- В описании события указывается преподаватель.
- Шкала прогресса и текстовый статус конвертации.
- При ошибке открывается окно с возможностью скопировать текст ошибки и отправить фидбэк разработчику.
- Выходной CSV-файл сохраняется с именем `Выбранная_группа_calendar.csv`.

---

## 📖 Инструкция по использованию

1. **Запуск программы**  
   Дважды щёлкните по файлу `.exe`.

2. **Выбор файла расписания**  
   Нажмите кнопку **«Обзор»**. Укажите Excel-файл (`.xls` или `.xlsx`) расписания, скачанный из сообщества **Факультет Социальных Технологий ВК**. После выбора программа откроет окно **«Выбор группы»**.

3. **Выбор группы**  
   В списке выберите свою группу (можно дважды щёлкнуть). Нажмите **«Выбрать»** или **«Отмена»**. Выбранная группа отобразится в главном окне.

4. **Выбор места сохранения**  
   Автоматически подставится путь рядом с исходным файлом. При необходимости нажмите **«Сохранить как...»** и укажите другую папку или имя файла.

5. **Конвертация**  
   Нажмите кнопку **«Конвертировать»**. Следите за шкалой прогресса. По окончании появится сообщение об успехе.

6. **Импорт в Google Календарь**  
   Откройте Google Календарь → **Настройки** → **Импорт и экспорт**. Выберите CSV-файл, укажите целевой календарь, нажмите **«Импортировать»**.

---

## ⚠️ Важное примечание

**Программа работает ТОЛЬКО с шаблоном расписаний, который публикуется в группе Факультета Социальных Технологий ВКонтакте.**  
Другие форматы Excel-файлов могут не поддерживаться.

## 💻 Минимальные системные требования

| Компонент                | Требование                                       |
|--------------------------|--------------------------------------------------|
| Операционная система     | Windows 7 / 8 / 10 / 11 (64-разрядная)           |

| Свободное место на диске | ~100 МБ                                          |
| Дополнительное ПО        | **Не требуется** (Microsoft Excel не нужен)      |

> Программа не требует установки Python, библиотек или Excel. Всё необходимое уже включено в исполняемый файл.
